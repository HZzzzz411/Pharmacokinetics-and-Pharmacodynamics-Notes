# Week5-9 知识点整理

## 一、一室模型

### 1.1 一室模型概述

#### 1.1.1 定义与特点
- **一室开放模型**: 假设身体可以被描述为一个单一、均匀的隔室（即一个隔室），并且药物可以进入和离开身体（即开放模型）
- **最简单给药方式**: 将全部药物以快速静脉注射方式给药，也称为静脉推注(IV Bolus)
- **假设条件**: 
  - 药物被瞬间注入体内
  - 药物被快速且瞬间分布到全身
  - 药物在进入体内后立即开始消除

#### 1.1.2 一室模型与两室模型对比
- **一室模型**: 药物被瞬间且均匀分布到一个单一（一）个流体隔室中
- **两室模型**: 将身体分为中心室（血浆/血液）和外周室（组织）
- **适用情况**: 
  - 一室模型: 当药物与血浆蛋白高度结合或分子太大而无法离开血管隔室时
  - 两室模型: 当药物在不同组织中的分布速度不同时

### 1.2 消除速率常数(k)

#### 1.2.1 基本概念
- 一级消除过程: 消除速率与体内药物量或浓度成正比
- 消除速率常数k表示代谢和排泄的一级速率过程之和

#### 1.2.2 数学表达
- 速率表达式:  $\frac{dD}{dt} = -kD $ 
- 积分后:  $\log_{10}D = -\frac{kt}{2.3} + \log_{10}D_0 $ 
- 其中D表示t时刻体内药物量，D₀表示t=0时体内的药物量

#### 1.2.3 消除半衰期(t₁/₂)
- $t_{1/2} = \frac{0.693}{k}$
- 代表药物浓度减少一半所需的时间

### 1.3 表观分布容积(Vd)

#### 1.3.1 基本概念
- 体积反映药物在体内似乎占据的空间
- Vd是一个理论体积，不是真实的生理体积
- $D = V_dC_p$，其中D是体内药物量，Cp是血浆药物浓度

#### 1.3.2 计算方法
- **方法1**:  $V_d = \frac{Dose}{C_{p0}} = \frac{D}{C_{p0}}$
  - 其中Cp0表示t=0时药物在体内平衡后的瞬时药物浓度
- **方法2（非隔室法）**:  $V_d = \frac{D_0}{k \cdot AUC_{0 \to \infty}}$
  - $AUC_{0 \to \infty} = \int_{0}^{\infty} C_p dt$

#### 1.3.3 临床意义
- Vd反映药物在体内分布的广泛程度
- **Vd < 体质量**: 药物与血浆蛋白高度结合或分子太大而无法离开血管隔室
- **Vd > 体质量**: 药物在周围组织和器官中浓缩，体内Cp0很小

### 1.4 清除率(Cl)

#### 1.4.1 基本概念
- 清除率是指每单位时间内被清除药物的血浆流体体积
- $Cl = \frac{dD/dt}{C_p} = kV_d$

#### 1.4.2 临床意义
- 与半衰期关系:  $t_{1/2} = \frac{0.693(V_d)}{Cl}$
- 直接与全身暴露（AUC）相关，是临床上最有用的药动学参数
- 当药动学行为遵循线性药动学时，清除率是恒定的

#### 1.4.3 计算方法
- **方法1（隔室法）**:  $Cl = kV_d$
- **方法2（非隔室法）**:  $Cl = \frac{Dose}{AUC_{0 \to \infty}}$

### 1.5 一室模型药动学方程

#### 1.5.1 血浆浓度-时间关系
- $C_p = C_{p0}e^{-kt}$
- 其中Cp0是t=0时血浆中的药物浓度

#### 1.5.2 一室模型AUC计算
- $AUC_{0 \to \infty} = \frac{D_0}{kV_d}$

### 1.6 多剂量给药方案

#### 1.6.1 多剂量给药原因
- 单次给药后，血浆药物水平先上升到高于最小有效浓度(MEC)，然后下降到低于MEC，导致治疗效果下降
- 多剂量或静脉输注方案用于维持血浆药物水平在治疗窗口内

#### 1.6.2 多剂量给药方案的关键参数
- 药物剂量的大小
- τ，药物给药频率（即给药间隔，即两次给药之间的时间）

#### 1.6.3 稳态时的血浆浓度
- $C_{max,ss} = \frac{D_0}{V_d(1 - e^{-k\tau})} = \frac{C_{p0}}{1 - e^{-k\tau}}$
- $C_{min,ss} = C_{max,ss}e^{-k\tau}$
- $C_{av,ss} = \frac{D_0}{V_dk\tau}$

#### 1.6.4 药物积累指数
- $R = \frac{C_{max,ss}}{C_{max,1}} = \frac{1}{1 - e^{-k\tau}}$
- R > 1表示存在药物积累

#### 1.6.5 任意时间点的药物浓度
- $C_p(t) = \frac{D_0}{V_d} \left( \frac{1 - e^{-nk\tau}}{1 - e^{-k\tau}} \right) e^{-kt'}$
- 其中t' = t - (n-1)τ，0 ≤ t' < τ

#### 1.6.6 给药间隔内的AUC
- $AUC_{(n-1)\tau \to n\tau} = \frac{D_0}{kV_d}(1 - e^{-nk\tau})$
- $AUC_{ss} = \frac{D_0}{kV_d}$

## 二、多室模型

### 2.1 两室模型

#### 2.1.1 基本概念
- **两室模型**: 将身体分为中心室和外周室
- **中心室**: 血流量最高的组织中的药物与血浆中的药物快速平衡的区域
- **外周室**: 由血流量较低且对药物亲和力不同的组织群组成的区域
- k01: 从贮库室到中心室的速率常数
- k10: 从中心室到体外的消除速率常数
- k12: 从中心室到外周室的分布速率常数
- k21: 从外周室到中心室的分布速率常数


#### 2.1.2 两室模型的数学描述
- 中心室:  $\frac{dx_1(t)}{dt} = -(k_{10} + k_{12})x_1(t) + k_{21}x_2(t)$
- 外周室:  $\frac{dx_2(t)}{dt} = k_{12}x_1(t) - k_{21}x_2(t)$

#### 2.1.3 两室模型的解
- $x_1(t) = \frac{D_0(\alpha - k_{21})}{(\alpha - \beta)}e^{-\alpha t} + \frac{D_0(k_{21} - \beta)}{(\alpha - \beta)}e^{-\beta t}$
- $x_2(t) = \frac{D_0k_{12}}{\alpha - \beta}(e^{-\beta t} - e^{-\alpha t})$
- 其中α和β是混合常数，满足: 
  - $\alpha\beta = k_{10}k_{21}$
  - $\alpha + \beta = k_{10} + k_{12} + k_{21}$
  - 因此 $\alpha, \beta = \frac{(k_{10} + k_{12} + k_{21}) \pm \sqrt{(k_{10} + k_{12} + k_{21})^2 - 4k_{10}k_{21}}}{2}$
  

#### 2.1.4 血浆浓度方程
- $C_1(t) = Ae^{-\alpha t} + Be^{-\beta t}$
- $A = \frac{D_0(\alpha - k_{21})}{V_c(\alpha - \beta)}$
- $B = \frac{D_0(k_{21} - \beta)}{V_c(\alpha - \beta)}$
- 外周室浓度方程 $C_2(t) = \frac{D_0 k_{12}}{V_t (\alpha - \beta)} (e^{-\beta t} - e^{-\alpha t})$
- A和B是曲线上每个指数段的y轴截距，即: $C_1(0) = A + B$ ;可通过残差法图形化地获得
- A和B是混合常数，没有实际的生理意义

### 2.2 残差法

#### 2.2.1 基本原理
- 用于将曲线分解为各种指数项的技术
- 通过将曲线外推到y轴得到B值，然后从原始数据中减去外推线的值，得到α相

#### 2.2.2 具体步骤
1. 使用较晚的时间点（β相），绘制 ln Cp vs. t
2. 从斜率得到β:  $Slope = -\beta/2.3$
3. 外推得到B值
4. 从原始数据中减去 $Be^{-\beta t}$，得到残差
5. 对残差取对数，绘制 ln(残差) vs. t，得到α: $Slope = -\alpha/2.3$
6. 外推得到A值

#### 2.2.3 残差法实例
- **数据**: 100 mg药物快速静脉注射给70 kg成人，血浆浓度数据如下: 
  | 时间(小时) | 血浆浓度(μg/mL) |
  | --- | --- |
  | 0.25 | 43.00 |
  | 0.5 | 32.00 |
  | 1.0 | 20.00 |
  | 1.5 | 14.00 |
  | 2.0 | 11.00 |
  | 4.0 | 6.50 |
  | 8.0 | 2.80 |
  | 12.0 | 1.20 |
  | 16.0 | 0.52 |

- **β相计算**: 
  - 选择较晚时间点(4-16小时)进行分析
  - 绘制ln Cp vs. t，得到直线
  - 从斜率计算β: $Slope = -0.21 h^{-1}$ → $β = 0.21 h^{-1}$
  - 外推到t=0，得到B = 15.06 μg/mL

- **α相计算**: 
  - 从原始数据中减去 $15e^{-0.21t}$
  - 对于t=0.25, 0.5, 1.0, 1.5: 
    - $43 - 15e^{-0.21 \times 0.25} \approx 28.77$
    - $32 - 15e^{-0.21 \times 0.5} \approx 18.50$
    - $20 - 15e^{-0.21 \times 1.0} \approx 7.84$
    - $14 - 15e^{-0.21 \times 1.5} \approx 3.05$
  - 绘制ln(残差) vs. t，得到斜率: $Slope \approx -1.8 h^{-1}$ → $α = 1.8 h^{-1}$
  - 外推得到A = 45.6 μg/mL

- **最终方程**: 
  - $C_1 = 45.6e^{-1.8t} + 15.06e^{-0.21t}$

### 2.3 两室模型参数计算

#### 2.3.1 中心室体积(Vc)
-  $V_c = \frac{D_0}{A + B}$
   - **详细推导**: 在零时间(t=0)，体内的全部药物都在中心室，C₁⁰ = A + B
      -  $C_{10} = A + B$（因为 $C_1(t) = Ae^{-\alpha t} + Be^{-\beta t}$，当t=0时， $C_1(0) = A + B$）
      -  $D_0 = V_c C_{10} = V_c (A + B)$
      -  $V_c = \frac{D_0}{A + B}$
-  $V_c = \frac{D_0}{k \cdot AUC_{0 \to \infty}}$

#### 2.3.2 消除参数计算
- $k_{10} = \frac{\alpha\beta(A + B)}{A\beta + B\alpha}$
- $k_{12} = \frac{AB(\beta - \alpha)^2}{(A + B)(A\beta + B\alpha)}$
- $k_{21} = \frac{A\beta + B\alpha}{A + B}$

#### 2.3.3 体积参数计算
- Vss的计算: 
  - 稳态时， $D_t = \frac{k_{12}V_p C_p}{k_{21}}$
  - 体内总药量 =  $D_t + D_p = \frac{k_{12}V_p C_p}{k_{21}} + V_p C_p$
  - 稳态时表观分布容积: $(V_D)_{ss} = \frac{D_t + D_p}{C_p} = V_p + \frac{k_{12}}{k_{21}}V_p$
  - 其中 $D_t$是组织室中的药物量， $D_p$是中心室中的药物量
-  $V_t = \frac{V_c k_{12}}{k_{21}}$
-  $(V_D)_{ss} = V_c + \frac{k_{12}}{k_{21}}V_c$ 

#### 2.3.4 面积法分布容积

-   $(V_D)_{\beta} = \frac{D_0}{\beta \cdot AUC_{0 \to \infty}} = \frac{Cl}{\beta} = \frac{kV_c}{\beta} $ 

### 2.4 三室模型

#### 2.4.1 基本概念
- 两室模型的扩展，增加了一个深组织室
- 药物最快分布到高灌注的中心室，较慢分布到第二个或组织室，非常缓慢地分布到第三个或深组织室
- 深组织室可能包含灌注不良的组织（如骨和脂肪）或表示组织中紧密结合的药物

#### 2.4.2 三室模型方程
- $C_p = Ae^{-\alpha t} + Be^{-\beta t} + Ce^{-\delta t}$
- 其中α、β、δ分别为中心室、组织室和深组织室的一阶速率常数

#### 2.4.3 三室模型微分方程
- $\frac{dx_p}{dt} = -(k_{10} + k_{12} + k_{13})x_p + k_{21}x_t + k_{31}x_{dt}$
- $\frac{dx_t}{dt} = k_{12}x_p - k_{21}x_t$
- $\frac{dx_{dt}}{dt} = k_{13}x_p - k_{31}x_{dt}$
- 其中 $x_p$ 是中心室中的药物量，$x_t$ 是组织室中的药物量, $x_{dt}$ 是深组织室中的药物量

#### 2.4.4 静脉输注给药
- 静脉输注的微分方程为:
 
  $
  \begin{align*}
  \frac{dx_1(t)}{dt} &= k_0 - (k_{10} + k_{12})x_1(t) + k_{21}x_2(t) \\
  \frac{dx_2(t)}{dt} &= k_{12}x_1(t) - k_{21}x_2(t)
  \end{align*}
  $
  
  其中k₀是输注速率


## 三、非线性药动学

### 3.1 非线性药动学的来源

#### 3.1.1 饱和代谢（容量限制代谢）
- 酶（如CYP450、醇脱氢酶）在高药物浓度下变得饱和
- 一旦饱和，消除从一级（比例）转变为零级（恒定速率）
- 例子: 苯妥英、乙醇、茶碱、高剂量水杨酸盐

#### 3.1.2 饱和（主动）转运
- 主动转运体（如肠或肾）在高剂量下达到容量，限制吸收或排泄

#### 3.1.3 饱和血浆蛋白结合
- 在高药物浓度下，血浆蛋白结合位点变得饱和，不成比例地增加游离药物水平
- 例子: 高剂量水杨酸盐表现出可饱和的白蛋白结合

#### 3.1.4 酶诱导或抑制
- 诱导增加代谢，随着时间的推移降低血浆药物水平
- 抑制减慢代谢，增加药物浓度
- 可导致时间依赖性非线性

#### 3.1.5 生理约束
- 器官血流变化（如肝脏、肾脏）
- 高剂量导致的生理变化（如pH变化、体液分布）
- 
#### 3.1.6 非线性PK的判断方法
- **剂量依赖性PK的判断**: 
  1. 通过静脉给药不同剂量，获得血浆药物浓度-时间曲线
  2. 如果药物遵循剂量独立动力学，则不同剂量的血浆浓度-时间曲线应具有平行斜率
  3. 或者，不同剂量下血浆浓度-时间曲线的AUC与剂量应呈线性关系
  4. 通过绘制AUC与剂量的曲线，如果该曲线是线性的，则药物遵循线性药动学；如果曲线是弯曲的，则表明存在非线性药动学

### 3.2 Michaelis-Menten动力学

#### 3.2.1 基本方程
- $-\frac{dC_p}{dt} = \frac{V_{max}C_p}{K_m + C_p}$
- 其中Vmax是最大消除速率，Km是Michaelis常数

#### 3.2.2 K_m的定义
- $K_m = \frac{k_{-1} + k_2}{k_1}$
- K_m不是消除常数，而是混合速率常数, 反映酶系统的容量
-  $E + S \xrightleftharpoons[k_{-1}]{k_1} ES \xrightarrow{k_2} E + P$
- **k1**: 酶-底物(ES)复合物形成的速率
- **k-1**: ES解离回E+S的速率
- **k2（或 $k_cat$）**: 催化速率常数（产物形成）
- **E**: 酶
- **S**: 底物
- **ES**: 酶-底物复合物
- **P**: 产物

#### 3.2.3 消除速率特性
- 当Cp << Km时: 一级消除
  - $-\frac{dC_p}{dt} = \frac{V_{max}C_p}{K_m} = kC_p$
- 当Cp >> Km时: 零级消除
  - $-\frac{dC_p}{dt} = V_{max}$

#### 3.2.4 临床意义
- 非线性药动学行为导致药物浓度-时间曲线的非线性
- 与剂量相关的药动学特性

### 3.3 非线性药动学的模型

#### 3.3.1 Emax药物-浓度效应模型
- $E = E_{max} \cdot \frac{C}{EC_{50} + C}$
- Emax: 药物可产生的最大可能效应
- EC50: 产生50% Emax的药物浓度

#### 3.3.2 S形Emax药物-浓度效应模型
- $E = E_{max} \cdot \frac{C^{\gamma}}{EC_{50}^{\gamma} + C^{\gamma}}$
- γ: Hill系数，描述效应-浓度关系的陡度

#### 3.3.3 线性浓度效应模型
- $E = S \cdot C$
- S: 线性浓度-效应关系线的斜率

#### 3.3.4 加性和比例药物效应模型
- **加性模型**: $R(t) = R(0) + E$
- **比例模型**: $R(t) = R(0) \cdot (1 + E)$

### 3.4 非线性药动学参数计算

#### 3.4.1 一室模型（静脉推注）
- **任意时间t的Cp计算公式**:
 $C_p(t) = \frac{D_0}{V_d} \left( \frac{1 - e^{-nk\tau}}{1 - e^{-k\tau}} \right) e^{-kt'}$
  
  其中t' = t - (n-1)τ，0 ≤ t' < τ

- **药物剂量从初始量降至某一量所需时间的公式**:
 $ t = \frac{1}{V_{max}}(D_0 - D) + K_m \ln \frac{D_0}{D}$
  
  其中D₀是初始剂量，D是任意时间t的剂量

- **双倒数图**: 
  - 通过绘制1/v vs. 1/Cp，可得到直线
  - 直线的y截距为1/Vmax
  - 斜率为Km/Vmax
  -  $1/v = \frac{K_m}{V_{max}} \frac{1}{C_p} + \frac{1}{V_{max}}$

#### 3.4.2 体内参数确定
- $R = \frac{V_{max}C_{ss}}{K_m + C_{ss}}$
- 通过两组不同剂量数据，可以解出Km和Vmax
- $K_m = \frac{R_2 - R_1}{\frac{R_1}{C_1} - \frac{R_2}{C_2}}$
- 其中R₁和R₂是两个不同给药方案的给药速率，C₁和C₂是相应的稳态血浆药物浓度


#### 3.4.3 混合药物消除
- 药物可能通过平行途径代谢为几种不同的代谢物
- 例如，水杨酸钠被代谢为葡萄糖醛酸和甘氨酸结合物（马尿酸）
- 甘氨酸结合物的形成速率受甘氨酸可用量的限制
- 描述静脉推注后由一级和Michaelis-Menten动力学同时消除的药物的方程为:
   $ \frac{dC_p}{dt} = kC_p + \frac{V_{max}C_p}{K_m + C_p} $
  
  其中k是一级速率常数，表示所有一级消除过程的总和

#### 3.4.4 零级输入和非线性消除
- **零级输入**: 通常例子是恒定静脉输注
- **非线性消除方程**:
   $ \frac{dC_p}{dt} = \frac{k_0}{V} - \frac{V_{max}C_p}{K_m + C_p} $
  
  其中k₀是输注速率，V是表观分布容积

#### 3.4.5 一级吸收和非线性消除
- **一级吸收和非线性消除**: 
   $ \frac{dC_p}{dt} = k_a C_{gi} e^{-k_a t} - \frac{V_{max}C_p}{K_m + C_p}$
  
  其中kₐ是一级吸收速率常数，C_{gi}是胃肠道中的浓度

- **一级吸收和混合消除**: 
   $ \frac{dC_p}{dt} = k_a C_{gi} e^{-k_a t} - \frac{V_{max}C_p}{K_m + C_p} - kC_p $
  
  其中k是一级消除速率常数

#### 3.4.6 由于药物-蛋白结合引起的非线性药动学
- **游离药物浓度**: 
  $ C_f = C_p(1 - \text{结合分数})$
  
  对于任何蛋白结合药物，游离药物浓度(Cf)将始终小于总药物浓度(Cp)

- **一室模型药物与蛋白结合**: 
  - 该单室包含游离药物和结合药物，它们通过速率常数k₁和k₂动态相互转换
  - 药物消除仅与游离药物一起发生，以一级速率发生
  - 结合药物不被消除

- **药物-蛋白结合的数学表达**: 
   $C_b = \frac{(k_1/k_2)C_f}{1 + (k_1/k_2)C_f}$
  
  该方程可以重排为: 
   $ C_b = \frac{P C_f}{k_d + C_f} = C_p - C_f $
  
  解Cf: 
   $ C_f = \frac{1}{2} \left[ -P + k_d - C_p + \sqrt{(P + k_d - C_p)^2 + 4k_dC_p} \right]$

- **消除速率方程**: 
   $ \frac{dC_p}{dt} = -k C_f $ 
   $\frac{dC_p}{dt} = -k \cdot \frac{1}{2} \left[ -P + k_d - C_p + \sqrt{(P + k_d - C_p)^2 + 4k_dC_p} \right]$
  
  该方程不容易积分，但可以使用数值方法求解

- **临床意义**: 
  - 非线性药动学发生在高剂量下
  - 因为高剂量下有更多的游离药物可用，初始药物消除发生得更快
  - 对于表现出非线性药动学的药物，游离药物浓度可能首先缓慢增加，但当药物剂量超过蛋白结合饱和点时，游离血浆药物浓度可能突然上升


### 3.5 对乙酰氨基酚（APAP）实例

#### 3.5.1 药代动力学特征
- 在治疗剂量后，APAP主要转化为药理学上无活性的葡萄糖醛酸（52-57%）和硫酸（30-44%）结合物
- 小部分被氧化为活性代谢物NAPQI（5-10%）
- 高毒性剂量后，葡萄糖醛酸化也会饱和，更高比例的药物以未改变的形式排泄（约10%）并被氧化为NAPQI（>15%）

#### 3.5.2 非线性PK下的浓度-时间曲线
- 当给予大剂量时，得到的曲线在初始缓慢消除阶段后，在较低的血液浓度下表现出更快速的消除（曲线A）
- 对于小剂量药物，观察到表观第一级动力学（曲线B）

## 四、清除率

### 4.1 清除率概述

#### 4.1.1 基本概念
- 清除率是描述药物从体内或从单个器官清除的过程
- 清除率可以定义为单位时间内从体内清除的药物体积
-  $Cl = \frac{dD/dt}{C_p}$

#### 4.1.2 清除率与半衰期的关系
-  $t_{1/2} = \frac{0.693(V_d)}{Cl}$
- 清除率直接与全身暴露（AUC）相关

#### 4.1.3 临床意义
- 清除率比半衰期更重要，因为它直接与药物的全身暴露相关
- 清除率考虑了所有药物消除过程，无论其机制如何
- 当药物的药动学行为遵循线性药动学时，清除率是恒定的，而药物消除速率则不是

### 4.2 清除率的计算方法

#### 4.2.1 隔室法
- 一室模型:  $Cl = kV_d$
- 多室模型（例如，其中总分布容积Vss包括中央分布容积Vc和一个或多个外周分布容积），药物的全身清除率将是中央隔室消除速率常数(k10)和Vc的乘积:  $Cl = k_{10} \times V_c$

#### 4.2.2 非隔室法
- $Cl = \frac{Dose}{AUC_{0 \to \infty}}$

#### 4.2.3 混合方法
-  $Cl = \frac{D_0}{k \cdot AUC_{0 \to \infty}}$
  - 该方法将隔室方法和非隔室方法结合起来
  - 通过计算AUC₀→∞和k，可以估计Vd
  - k可以通过半衰期计算:  $k = \frac{0.693}{t_{1/2}}$
  - AUC₀→∞通过梯形法则计算
  - 该方法的优点是可以避免对隔室模型做出假设，同时提供对药物分布和消除的估计

### 4.3 清除率的临床应用

#### 4.3.1 肾功能影响
- 肾功能受损时，清除率降低
- 例如: 患者A（正常肾功能）和患者B（肾功能受损）的清除率差异

#### 4.3.2 肾功能受损患者的剂量调整
- 肾功能受损患者的AUC更高，MRT更长
- Vss通常不变，表明分布不受影响
- 需要剂量调整（例如，剂量减少约35%或给药间隔延长）

## 五、非隔室分析

### 5.1 非隔室分析概述

#### 5.1.1 基本概念
- 不指定特定隔室模型来描述药物药动学
- 通过统计矩理论进行计算
- 假设: 药物显示线性药动学，药物从测量的池（如血浆）中清除

#### 5.1.2 适用条件
- "丰富数据": 与单次给药相关的至少12个不同的浓度时间点
- 任何少于这些的数据可能在使用非隔室方法时对药动学参数提供不准确的估计

### 5.2 统计矩理论

#### 5.2.1 零矩（AUC）
-  $AUC_{0 \to \infty} = \int_{0}^{\infty} C_p(t) dt$
- 通过梯形法则计算: 
  -  $AUC_{t_n \to t_{n+1}} = \frac{C_{n+1} + C_n}{2} (t_{n+1} - t_n)$
  -  $ AUC_{0 \to \infty} = \sum AUC_{t_n \to t_{n+1}} + \frac{C_t}{\lambda_z}$
- **λz的含义**: λz是终末相的消除速率常数，代表药物在终末相的消除速率


#### 5.2.2 一矩（AUMC）
-  $AUMC_{0 \to \infty} = \int_{0}^{\infty} t \cdot C_p(t) dt$
- 通过梯形法则计算: 
  -  $AUMC_{t_n \to t_{n+1}} = \frac{t_{n+1}C_{n+1} + t_nC_n}{2} (t_{n+1} - t_n)$
  -  $ AUMC_{0 \to \infty} = AUMC_{0 \to t} + \frac{C_t \times t}{\lambda_z} + \frac{C_t}{\lambda_z^2} $

#### 5.2.3 平均滞留时间（MRT）
- **MRT公式**: 
   $ MRT = \frac{AUMC_{0 \to \infty}}{AUC_{0 \to \infty}} - \frac{\text{Duration infusion}}{2}$
  
  其中Duration是药物输注的持续时间

- **MAT（平均吸收时间）**: 平均吸收时间，或药物分子被吸收到全身循环所需的平均时间
- **MTT（平均转运时间）**: 平均转运时间，表示总转运时间，即MAT和MRT的总和
    $ MTT = MAT + MRT $
  
  - 当药物通过血管外给药（例如口服给药）时，AUMC/AUC比率不仅反映分子一旦进入全身循环的滞留时间（MRT），还反映它们被吸收的时间持续期
  - MTT = MAT + MRT

#### 5.2.4 平均转运时间、平均吸收时间与平均溶解时间
- **静脉给药后**，MRT基本上反映了体内的消除过程，因此反映了分子在全身循环中滞留的时间
- **血管外给药后**，AUMC/AUC比率不仅反映分子一旦进入全身循环的滞留时间（MRT），还反映它们被吸收的时间持续期
- **MTT**:  $MTT = \frac{AUMC_{0 \to \infty}}{AUC_{0 \to \infty}}$
- **MRT**:  $MRT = \frac{AUMC_{0 \to \infty}}{AUC_{0 \to \infty}}$
- **MAT**:  $MAT = MTT - MRT$


### 5.3 非隔室分析计算实例

#### 5.3.1 非隔室分析Q1
- **MRT**:  $MRT = \frac{AUMC}{AUC}$
- **Cl**:  $Cl = \frac{Dose}{AUC}$
- **Vss**:  $V_{ss} = Cl \times MRT$

### 5.3 其他PK参数

#### 5.3.1 清除率(Cl)
-  $Cl = \frac{Dose}{AUC_{0 \to \infty}}$

#### 5.3.2 总分布容积(Vss)
-  $V_{ss} = Cl \times MRT$
- **Vss的计算**: 在单次静脉给药后，Vss可以计算为Cl × MRT
  - Cl: 清除率
  - MRT: 平均滞留时间

#### 5.3.3 案例（补充详细内容）
- **案例**: 抗生素以1000 mg静脉推注剂量给予两名受试者。假设该药物的药动学被一室模型很好地描述。该药物的分布容积为10 L，并遵循一室模型，消除常数(λz)在两名受试者中分别为(1) 0.1 h⁻¹和(2) 0.2 h⁻¹。假设每名受试者在时间零的浓度为100 mg/L。
- **计算**: 
  1. 通过梯形法则计算每个受试者的AUC₀→t和AUMC₀→t
  2. 计算AUC₀→∞: 
      $AUC_{0 \to \infty} = AUC_{0 \to t} + \frac{C_t}{\lambda_z}$
     
     - 受试者1: $AUC_{0 \to \infty} = 961.605 + \frac{4.979}{0.1} = 1011.395$ mg·h/L
     - 受试者2: $AUC_{0 \to \infty} = 509.243 + \frac{0.248}{0.2} = 510.483$ mg·h/L
  4. 计算Cl:  $Cl = \frac{1000}{1011.395} = 0.99$ L/h（受试者1）
      $Cl = \frac{1000}{510.483} = 1.96$ L/h（受试者2）
  5. 计算AUMC₀→∞:
     $ AUMC_{0 \to \infty} = AUMC_{0 \to t} + \frac{C_t \times t}{\lambda_z} + \frac{C_t}{\lambda_z^2}$
     
     - 受试者1: $AUMC_{0 \to \infty} = 1383.135 + \frac{149.37}{0.1} + \frac{4.979}{0.1^2} = 9963.4$（受试者1）
     - 受试者2: $AUMC_{0 \to \infty} = 525.308 + \frac{7.44}{0.2} + \frac{0.248}{0.2^2} = 2526.9$（受试者2）
  7. 计算MRT:
      $ MRT = \frac{AUMC_{0 \to \infty}}{AUC_{0 \to \infty}} - \frac{\text{Duration infusion}}{2}$
     
     - 受试者1: $MRT = \frac{9963.4}{1011.395} = 9.85$ h
     - 受试者2: $MRT = \frac{2526.9}{510.483} = 4.95$ h

## 六、药动学与药效学

### 6.1 PK/PD在药物开发中的作用

#### 6.1.1 药物开发过程
- **临床前测试**: 在动物模型中测试生物活性
- **1期研究**: 在健康志愿者中研究PK和耐受性
- **2期研究**: 在患者中评估治疗效益
- **3期研究**: 用于药物批准的关键性试验

#### 6.1.2 PK/PD在药物开发中的应用
- 通过PK-PD信息构建剂量-暴露-反应关系
- 为3期研究设计合理的给药策略

### 6.2 药物-受体相互作用

#### 6.2.1 药物-受体相互作用
- 药物通过与特定细胞分子称为受体的相互作用产生效应
- 通常遵循质量作用定律，可根据受体占有率理论描述

#### 6.2.2 药物-受体相互作用模型
- $Drug + Receptor \rightleftharpoons [Drug-receptor complex] \rightarrow Response$
  其中方括号[]表示摩尔浓度
- 该模型假设: 
  1. 药物分子与受体分子以双分子结合
  2. 药物与受体的结合是完全可逆的
  3. 基本模型假设一种类型的受体结合位点
  4. 一个受体位点上药物分子的占有率不会改变更多药物分子与额外受体位点结合的亲和力
  5. 每个受体对药物分子具有相等的亲和力

#### 6.2.3 受体占有率理论
- **受体占有率**: $f_{occ} = \frac{[RC]}{R_{max}}$
- **E与f_occ的关系**: $E = E_{max} \cdot \frac{[RC]}{R_{max}}$
- **[RC]的计算**: 
   $ [RC] = \frac{R_{max} C}{K_d + C}$
  其中Kd是平衡解离常数，$K_d = \frac{k_{off}}{k_{on}}$

- **K_off和K_on的含义**: 
  - $k_{on}$: 酶-底物(ES)复合物形成的速率常数
  - $k_{off}$: ES解离回E+S的速率常数

- **EC50和Kd**: 在E的计算公式中，EC50和Kd可以互换，因为EC50 = Kd

#### 6.2.4 与药物-受体相互作用相关的分类
- **激动剂**: 与受体结合并激活受体产生最大效应
- **部分激动剂**: 与受体结合但产生低于最大效应
- **反向激动剂**: 与受体结合并抑制受体活性
- **拮抗剂**: 与受体结合但不激活受体，阻止激动剂的作用

#### 6.2.5 生物标志物
- **生物标志物**: 可以定量测量的参数，代表体内健康或病理过程
- **替代终点**: 打算替代临床有意义终点的生物标志物
- **临床终点**: 与患者感觉、功能或生存的临床有意义度量相关

### 6.3 药动学-药效学模型

#### 6.3.1 PK-PD框架
- **PK部分**: $C_p = f(\theta_{PK}, X, t)$
- **PD部分**: $R = f(\theta_{PD}, C_p \text{ or } C_{eff}, Z)$
- 其中: 
  - $\theta_{PK}$: 基本PK参数（Cl，Vd）
  - $\theta_{PD}$: 基本PD参数（E_max，EC50）
  - X: 受试者变量（剂量，给药方案）
  - Z: 药物无关系统参数

#### 6.3.2 非隔室分析
- **PK-PD指数**: 
  - **Cmax/MIC**: 峰血浆药物浓度与最低抑制浓度的比值
  - **AUC/MIC**: 曲线下面积与最低抑制浓度的比值
  - **T > MIC**: 系统药物浓度保持在病原体的药物MIC以上的持续时间百分比

- **PK-PD指数的含义**: 
  - **Cmax/MIC**: 表示峰值浓度与MIC的比值，对氨基糖苷类和氟喹诺酮类抗生素的治疗疗效相关
  - **AUC/MIC**: 表示总暴露与MIC的比值，与β-内酰胺类抗生素和大环内酯类的治疗疗效相关
  - **T > MIC**: 表示药物浓度在MIC以上的持续时间，与β-内酰胺类抗生素和大环内酯类的治疗疗效相关

#### 6.3.3 非线性PK-PD关系
- **浓度依赖性杀灭**: 与药物浓度高于病原体MIC时杀灭速率和程度增加相关
- **时间依赖性杀灭**: 在系统浓度超过阈值或MIC时产生更高的系统浓度，并不会导致微生物杀灭速率的成比例增加

### 6.4 药物-浓度效应模型

#### 6.4.1 Emax药物-浓度效应模型
- $E = E_{max} \cdot \frac{C}{EC_{50} + C}$
- Emax: 药物可产生的最大可能效应
- EC50: 产生50% Emax的药物浓度
- 适用于描述大多数药物的剂量-效应关系

#### 6.4.2 S形Emax药物-浓度效应模型
- $E = E_{max} \cdot \frac{C^{\gamma}}{EC_{50}^{\gamma} + C^{\gamma}}$
- 其中γ为Hill系数，描述效应-浓度关系的陡度
- **Hill系数对曲线斜率的影响**: 
  - γ ≤ 1: 曲线较平缓
  - 1 < γ < 4: 曲线陡度逐渐增加
  - γ > 4: 曲线非常陡峭，表示全或无响应

- **Hill系数的临床意义**: 
  - 如果药物具有陡峭的浓度-效应关系（高γ值），那么错过一次剂量对受试者反应的影响可能更大


#### 6.4.3 线性浓度效应模型
- $E = S \cdot C$
- 适用于描述药物对心脏复极化的影响

#### 6.4.4 加性和比例药物效应模型
- **加性模型**: $R(t) = R(0) + E$
- **比例模型**: $R(t) = R(0) \cdot (1 + E)$

#### 6.4.5 比例Emax药物效应模型
- **刺激性**: $R(t) = R(0) \cdot \left(1 + \frac{E_{max} \cdot C}{EC_{50} + C}\right)$
- **抑制性**: $R(t) = R(0) \cdot \left(1 - \frac{I_{max} \cdot C}{IC_{50} + C}\right)$
  其中C是时间t的血浆药物浓度
  - I_max的通用表达式为: $I_{max} = \frac{R_0 - R_{max}}{R_0}$，其中R₀是时间t=0或基线时的反应，R_max是最大抑制反应

### 6.5 药物-浓度效应关系的临床应用

#### 6.5.1 临床剂量调整
- 通过PK-PD关系确定适当的剂量
- 通过PK参数预测血浆浓度-时间曲线
- 为不同患者群体（如肾功能受损患者）调整剂量

#### 6.5.2 临床终点与生物标志物
- 生物标志物: 可以定量测量的参数，代表体内健康或病理过程
- 替代终点: 打算替代临床有意义终点的生物标志物
- 临床终点: 与患者感觉、功能或生存相关的临床有意义度量

## 七、综合应用

### 7.1 药物剂量方案设计

#### 7.1.1 药物剂量方案设计原则
- 通过PK参数预测血浆浓度-时间曲线
- 考虑治疗窗口（MEC-MTC）内维持药物浓度
- 调整剂量大小和给药间隔

#### 7.1.2 剂量方案实例
- 例如: 1000 mg抗生素每6小时通过重复静脉注射给药
- 计算Cmax,ss、Cmin,ss和Cav,ss
- 评估药物在稳态下的浓度变化

### 7.2 遗漏剂量的处理

#### 7.2.1 遗漏剂量的影响
- 遗漏剂量会导致血浆浓度降低
- 需要计算遗漏剂量后的浓度

#### 7.2.2 遗漏剂量的计算
- $C_p = \frac{FD_0}{V_d} \left( \frac{1 - e^{-nk\tau}}{1 - e^{-k\tau}} \right) e^{-kt'} - \frac{FD_0}{V_d} e^{-k(t_{missed})} + \frac{FD_0}{V_d} e^{-k(t_{new})}$
  - $t_{missed}$: 遗漏剂量后的时间
  - $t_{new}$: 新的给药时间
- **案例**: 头孢菌素(k=0.2 h⁻¹, Vd=10 L)静脉多剂量给药；100 mg每6小时注射6次。第五次剂量被遗漏，第六次给药后4小时（即40小时后）的血浆药物浓度是多少？
  - 无遗漏剂量: $C_p = \frac{100}{10} \left( \frac{1 - e^{-6 \times 0.2 \times 6}}{1 - e^{-0.2 \times 6}} \right) e^{-0.2 \times 4} \approx 6.425 mg/L$
  - 遗漏第五次剂量: $t_{missed} = 6+4 = 10$小时
  - $C_p = \frac{100}{10} e^{-0.2 \times 10} \approx 1.353 mg/L$
  - 修正遗漏剂量后的药物浓度 = 6.425 - 1.353 = 5.072 mg/L


### 7.3 临床应用案例

#### 7.3.1 氨力农（Amrinone）案例（补充详细内容）
- **数据**: 75 mg单次静脉推注
- **参数**: $A=4.62\mu g/mL$，$B=0.64\mu g/mL$，$\alpha=8.94h^{-1}$，$\beta=0.19h^{-1}$
- **计算**: 
  - **中心室体积(Vc)**: $V_c = \frac{75,000}{4.62 + 0.64} = 14,259$ mL
  - **k10**: $k_{10} = \frac{\alpha\beta(A + B)}{A\beta + B\alpha} = \frac{8.94 \times 0.19 \times (4.62 + 0.64)}{4.62 \times 0.19 + 0.64 \times 8.94} = 1.35 h^{-1}$
  - **k12**: $k_{12} = \frac{AB(\beta - \alpha)^2}{(A + B)(A\beta + B\alpha)} = \frac{4.62 \times 0.64 \times (0.19 - 8.94)^2}{(4.62 + 0.64)(4.62 \times 0.19 + 0.64 \times 8.94)} = 6.52 h^{-1}$
  - **k21**: $k_{21} = \frac{A\beta + B\alpha}{A + B} = \frac{4.62 \times 0.19 + 0.64 \times 8.94}{4.62 + 0.64} = 1.25 h^{-1}$
  - **组织室体积(Vt)**: $V_t = \frac{V_c k_{12}}{k_{21}} = \frac{14,259 \times 6.52}{1.25} = 74,375$ mL
  - **稳态时的分布容积(VD)ss**: $(V_D)_{ss} = V_c + \frac{k_{12}}{k_{21}}V_c = 14,259 + 74,375 = 88,634$ mL
  - **清除率(Cl)**: $Cl = k_{10} \times V_c = 1.35 \times 14,259 = 19,250$ mL/h

#### 7.3.2 苯妥英案例（补充详细内容）
- **数据**: 150 mg/d和300 mg/d给药，稳态血浆药物浓度分别为8.6和25.1 mg/L
- **计算**: 
  - **Km和Vmax**:
     $ K_m = \frac{R_2 - R_1}{\frac{R_1}{C_1} - \frac{R_2}{C_2}} = \frac{300 - 150}{\frac{150}{8.6} - \frac{300}{25.1}} \approx 27.3 $
    
     $ V_{max} = \frac{150 \times (27.3 + 8.6)}{8.6} \approx 626 $
    
  - **达到11.7 mg/L稳态浓度所需的剂量**:
     $ R = \frac{V_{max}C_{ss}}{K_m + C_{ss}} \implies R = \frac{626 \times 11.7}{27.3 + 11.7} \approx 187.8 \text{ mg/d} $

## 八、总结

### 8.1 隔室模型与非隔室模型比较

| **特征** | **隔室模型** | **非隔室模型** |
|---------|------------|--------------|
| **基础** | 数学框架 | 统计矩理论 |
| **假设** | 指定隔室模型 | 无指定隔室模型 |
| **参数** | k, Vd, Cl | MRT, AUC, AUMC |
| **优势** | 预测特定时间浓度 | 无需模型假设 |
| **劣势** | 模型选择可能不准确 | 不能预测特定时间浓度 |
| **应用** | 药物开发早期 | 药物开发后期 |

### 8.2 药物-受体相互作用

- **药物-受体相互作用**: 药物通过与特定细胞分子称为受体的相互作用产生效应
- **受体占有率理论**: $f_{occ} = \frac{[RC]}{R_{max}}$
- **E与f_occ的关系**: $E = E_{max} \cdot f_{occ}$

### 8.3 药动学与药效学关系

- 药动学: 药物在体内的吸收、分布、代谢和排泄(ADME)
- 药效学: 药物在作用部位的效应
- PK/PD: 将药物暴露与治疗效果联系起来

### 8.4 药物开发中的PK/PD应用

- 优化剂量方案
- 评估治疗效果和安全性
- 预测药物相互作用
- 个体化给药

### 8.5 临床应用

- 肾功能受损患者的剂量调整
- 老年患者的剂量调整
- 基于PK/PD的个体化治疗

### 8.6 未来发展方向

- 个体化药动学建模
- 基于PK/PD的临床决策支持
- 基于药动学-药效学的药物开发

**重要公式汇总**

1. 一室模型: 
   - $C_p = C_{p0}e^{-kt}$
   - $t_{1/2} = \frac{0.693}{k}$
   - $V_d = \frac{Dose}{C_{p0}}$
   - $Cl = kV_d$
   - $AUC_{0 \to \infty} = \frac{D_0}{kV_d}$

2. 多剂量给药: 
   - $C_{max,ss} = \frac{D_0}{V_d(1 - e^{-k\tau})}$
   - $C_{min,ss} = C_{max,ss}e^{-k\tau}$
   - $C_{av,ss} = \frac{D_0}{V_dk\tau}$

3. 两室模型: 
   - $C_1(t) = Ae^{-\alpha t} + Be^{-\beta t}$
   - $V_c = \frac{D_0}{A + B}$
   - $k_{10} = \frac{\alpha\beta(A + B)}{A\beta + B\alpha}$
   - $k_{12} = \frac{AB(\beta - \alpha)^2}{(A + B)(A\beta + B\alpha)}$
   - $k_{21} = \frac{A\beta + B\alpha}{A + B}$
   - $A = \frac{D_0(\alpha - k_{21})}{V_c(\alpha - \beta)}$
   - $B = \frac{D_0(k_{21} - \beta)}{V_c(\alpha - \beta)}$

4. 非线性药动学: 
   - $-\frac{dC_p}{dt} = \frac{V_{max}C_p}{K_m + C_p}$
   - $t = \frac{1}{V_{max}}(D_0 - D) + K_m \ln \frac{D_0}{D}$
   - $R = \frac{V_{max}C_{ss}}{K_m + C_{ss}}$

5. 非隔室分析: 
   - $MRT = \frac{AUMC_{0 \to \infty}}{AUC_{0 \to \infty}}$
   - $Cl = \frac{Dose}{AUC_{0 \to \infty}}$
   - $V_{ss} = Cl \times MRT$

6. 药动学-药效学: 
   - $E = E_{max} \cdot \frac{C}{EC_{50} + C}$
   - $E = E_{max} \cdot \frac{C^{\gamma}}{EC_{50}^{\gamma} + C^{\gamma}}$
   - $E = S \cdot C$

7. 清除率: 
   - $Cl = \frac{Dose}{AUC_{0 \to \infty}}$
   - $t_{1/2} = \frac{0.693(V_d)}{Cl}$

8. AUC计算: 
   - $AUC_{t_n \to t_{n+1}} = \frac{C_{n+1} + C_n}{2} (t_{n+1} - t_n)$
   - $AUC_{0 \to \infty} = AUC_{0 \to t} + \frac{C_t}{\lambda_z}$

9. AUMC计算: 
   - $AUMC_{t_n \to t_{n+1}} = \frac{t_{n+1}C_{n+1} + t_nC_n}{2} (t_{n+1} - t_n)$
   - $AUMC_{0 \to \infty} = AUMC_{0 \to t} + \frac{C_t \times t}{\lambda_z} + \frac{C_t}{\lambda_z^2}$

10. 遗漏剂量: 
    - $C_p = \frac{FD_0}{V_d} \left( \frac{1 - e^{-nk\tau}}{1 - e^{-k\tau}} \right) e^{-kt'} - \frac{FD_0}{V_d} e^{-k(t_{missed})} + \frac{FD_0}{V_d} e^{-k(t_{new})}$
